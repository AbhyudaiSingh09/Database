
Introduction 

The Project helps in automating the transfer of CSV file data to respective tables
in MYSQL database on Omege server. The Project is written in Python and is using 
MYSQL-Connector to connect to the Database. 

The project has been done in parts for each table. 

Requirements
Python 
Code Editor
MYSQL Connector 

Installation 
Install Python 3.7 or above 
Install Code Editor
Install Virtualenv 

Configuration 
Create Virtualenv and activate virtualenv. 
Unzip the zip folder and open using the code Editor. 
Keep the .csv files within the unzipped folder


Execution 
Execute one function one at a time for each table.
